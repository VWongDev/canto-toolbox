# Agent Instructions

## Package Manager
Always use `pnpm`. Never use `npm` or `yarn`.

## Building
Run `pnpm build` to build the extension. See @.agents/dev-workflow.md for the full pipeline and incremental build shortcuts.

- Never edit `src/data/mandarin.json` or `src/data/cantonese.json` — they are generated by `pnpm build:dict`
- Never edit `build-tools/dist/` — it is generated by `pnpm build:scripts`
- If the build behaves unexpectedly, run `pnpm clean` then `pnpm build` from scratch

## Git Commits
All commits must follow the `type(domain): Description` format. Read and follow @.agents/git-conventions.md exactly.

## Codebase
Read @.agents/architecture.md to understand component responsibilities and data flow before modifying source files.
